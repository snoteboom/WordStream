Status

The FreeBSD/ppc port is still a Tier 2 platform.That means it is not fully supported by out security officer, release engineers and toolchain maintainers.

Latest News

  • 02 April, 2009:Support for the powerPC G5 has been committed to SVN.
  • 03 March, 2008: Support for Freescale power QUICC III MPC85XX family system-on-chip development boards has been committer into CVS.
  • 27 Febuary, 2008: FreeBSD 7.0 is the first release to officially support the FreeBSD/ppc prt.
